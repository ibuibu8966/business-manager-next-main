# Business Manager v1.0 - 機能概要

> 作成日: 2024年12月14日  
> バージョン: 1.0

## 📌 このアプリについて

**Business Manager** は、中小企業やフリーランス向けの**業務管理アプリケーション**です。

日々の業務で発生する以下を一元管理できます：
- タスク（やること）の管理
- 顧客対応の記録
- 売上・経費の記録
- お金の貸し借り管理

---

## 🎯 主な機能

### 1. ダッシュボード（トップページ）
ログイン後に表示されるメイン画面です。

| 表示内容 | 説明 |
|---------|------|
| 未完了タスク数 | 完了していないタスクの数 |
| 対応中チケット数 | 顧客対応が必要なチケット数 |
| 顧客数 | 登録されている顧客の総数 |
| 今月の利益 | 売上 - 経費 の計算結果 |
| 期限超過アラート | 期限を過ぎたタスクがあれば警告 |
| 通知（管理者のみ） | チームメンバーの更新情報 |

---

### 2. タスク管理 📋
やるべきことを管理する機能です。

**できること：**
- タスクの作成・編集・削除
- ステータス管理（未着手 → 進行中 → 完了）
- 優先度設定（高・中・低）
- 期限の設定
- リマインダー機能（指定日時まで非表示にできる）

**ステータスの流れ：**
```
未着手 → 進行中 → 完了
```

---

### 3. 顧客管理 👥
取引先や顧客の情報を管理します。

**登録できる情報：**
- 名前
- 電話番号
- メールアドレス
- 住所
- メモ

---

### 4. チケット管理 🎫
顧客からの問い合わせや依頼を「チケット」として管理します。

**ステータスの流れ：**
```
新規 → 対応中 → 保留 or 完了
```

**対応元（どこから来た依頼か）：**
- 電話
- メール
- Web
- その他

---

### 5. 経理（売上・支出管理） 💰
売上と経費を記録して、利益を把握します。

**取引の種類：**
- 売上（income）
- 経費（expense）

**カテゴリ例：**
- 売上: 売上、その他収入
- 経費: 外注費、広告費、交通費、消耗品、家賃、サブスク、人件費、その他経費

**固定費の自動登録：**
毎月発生する固定費（家賃など）を設定しておくと、自動で経費に追加されます。

---

### 6. 貸し借り管理 🤝
社内口座間や外部の人との金銭の貸し借りを記録します。

**取引の種類：**
- 貸し（lend）- お金を貸した
- 借り（borrow）- お金を借りた
- 返却（return）- 貸し借りを精算した

---

### 7. 事業管理 🏢
複数の事業を管理できます。各データ（タスク、取引など）を事業ごとに分類できます。

---

### 8. ユーザー管理 👤
システムを使うユーザーを管理します。

**権限：**
- 管理者（isAdmin: true）: 全機能にアクセス可能、通知を受け取れる
- 一般ユーザー: 基本機能のみ

---

## 🔔 通知機能

- **期限超過アラート**: 期限を過ぎたタスクをダッシュボードに表示
- **チーム更新通知（管理者向け）**: メンバーがタスクを更新した履歴を表示
- **LINE通知（準備中）**: 将来的にLINEへの通知も対応予定

---

## 🔐 ログイン

アプリを使うにはログインが必要です。

**初期アカウント：**
- メール: `admin@example.com`
- パスワード: `admin123`

> ⚠️ 本番運用時は必ずパスワードを変更してください
